Μαριαλένα μου, 💌

Στην πιο όμορφη και μοναδική γυναίκα που φωτίζει τη ζωή μου, 💖

Σήμερα θέλω να σου μιλήσω ανοιχτά, γιατί υπάρχουν πράγματα που πρέπει να ακούσεις. Εσύ είσαι ο λόγος που νιώθω τόσο τυχερός κάθε μέρα. Από τότε που ήρθες στη ζωή μου, έχω καταλάβει πόσο πολύ μπορεί να αλλάξει ένας άνθρωπος—για το καλύτερο. 🌹

Σε αγαπώ για τον γλυκό σου χαρακτήρα, που με κάνει να νιώθω ήρεμος και αγαπημένος. Σε αγαπώ για το γέλιο σου, που μου φέρνει χαρά ακόμα και όταν είμαι κουρασμένος. Σε αγαπώ για το χιούμορ σου, που με κάνει να γελάω χωρίς σταματημό. Και πάνω απ’ όλα, σε αγαπώ για τον τρόπο που νοιάζεσαι—για μένα, για τους ανθρώπους γύρω σου, και για τη ζωή γενικά. Ξέρω επίσης ότι πολλές φορές σου σπάω τα νεύρα (όπως κι εσύ τα δικά μου 😜), αλλά ακόμα κι αυτό είναι κομμάτι του πόσο ταιριάζουμε. 💑

Θέλω να ζήσουμε όσο περισσότερες στιγμές μπορούμε, να κάνουμε αμέτρητα ταξίδια μαζί και να δημιουργήσουμε μοναδικές αναμνήσεις που θα θυμόμαστε μια ζωή και όταν έρθει η στιγμή, θέλω να ταξιδέψουμε μαζί. Να δούμε μέρη που ονειρευόμαστε, να γελάσουμε, να φτιάξουμε αναμνήσεις. Αλλά ακόμα και αν αυτό αργήσει, δεν πειράζει. Για μένα, η μεγαλύτερή μου ευτυχία είναι να σε βλέπω κάθε μέρα. 🌍✈️
